# BaristaOS - Sistem Manajemen Kasir Cafe

Aplikasi kasir cafe berbasis C++ yang dibuat sebagai proyek akhir 
mata kuliah Praktikum Algoritma dan Pemrograman Lanjut.

## Fitur
- Lihat daftar menu cafe
- Kasir (transaksi pembelian)
- Cari menu
- Urutkan menu A-Z
- Stok persediaan
- Riwayat transaksi
- Laporan penjualan
- Tambah stok
- Baca file riwayat

## Konsep yang Diterapkan
- Struct
- Array satu dan dua dimensi
- Fungsi rekursif
- Sequential Search
- Bubble Sort
- File handling (ofstream & ifstream)
